What it does

everything you hate doing, automated

💬

Slack Triage

Reads your DMs, @mentions, and AI/ML topics across all channels. Drafts replies for your approval. Routes PR reviews and issues automatically.

🔍

PR Review

Critical-only code review on Bitbucket. No nitpicking, no style comments. Approves clean PRs silently. Detects re-reviews and resolved issues.

🛠️

Issue Handler

Investigates bugs (RCA), creates Jira tickets, brainstorms, writes plans, implements, tests, and creates PRs. 4-phase workflow with approval gates.

📋

Daily Planner

Morning planning with Todoist tasks from Jira + Slack + Bitbucket. Activity logging in Obsidian. End-of-day summaries. P1-P4 prioritization.

📱

Push Notifications

ntfy.sh pushes to your phone for urgent items and approval requests. Approve actions from anywhere with a simple API call.

⚡

Background Dispatch

Long tasks run in background with --bg flag. macOS sleep prevention via caffeinate. Check progress anytime with status command.

Get running in 60 seconds

1

Install

One command. Handles Python, Node, Claude CLI, deps, and PATH setup.

curl -fsSL https://raw.githubusercontent.com/predator-labs/work-agent/main/install.sh | bash
2

Configure

Add your Slack token, Bitbucket creds, Jira API key. Everything else is optional.

nano ~/work-agent/.env
3

Run

From any terminal, anywhere. Server mode or specific commands.

work-agent # start server
work-agent slack # triage messages
work-agent run-all # full cycle

Commands

type work-agent and go

work-agent Start the server
work-agent slack Triage Slack messages
work-agent review-pr <url> Review a PR
work-agent handle "..." Handle an issue
work-agent plan-day Morning planning
work-agent end-day End-of-day summary
work-agent run-all Full cycle
work-agent status Pending approvals
work-agent run-all --bg Background dispatch
